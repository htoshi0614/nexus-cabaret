# POS Start ログインマニュアル

## はじめに

POS Start はセキュリティのため、**ランダム英数字のパスワード**を使用しています。
パスワードはサーバー初回起動時に自動で生成され、2種類あります。

| パスワード | 用途 | 権限 |
|---|---|---|
| **オーナー用** | オーナー・管理者がログイン | 全機能（設定変更・売上確認・パスワード管理など） |
| **スタッフ用** | キャスト・スタッフがログイン | POS操作のみ（設定変更・売上確認は不可） |

---

## 初回ログイン手順

### ステップ1：パスワードを確認する

サーバーのフォルダ内にある **`.pos_credentials`** ファイルを開いてください。

```
# POS Start ログインパスワード（自動生成）
# このファイルを削除すると次回起動時に再生成されます
OWNER_PASSWORD=xxxxxxxx
STAFF_PASSWORD=xxxxxx
```

`OWNER_PASSWORD=` の右側がオーナー用、`STAFF_PASSWORD=` の右側がスタッフ用のパスワードです。

### ステップ2：ログイン画面を開く

1. ブラウザ（Chrome推奨）でPOS Startのアドレスにアクセスします
2. ページ下部の **「パスワードでログイン」** をタップします

### ステップ3：パスワードを入力する

1. パスワード欄に **オーナー用** または **スタッフ用** のパスワードを入力します
2. **「ログイン」** をタップします
3. POS画面が表示されればログイン成功です

---

## ログイン後にパスワードを確認・変更する

オーナーとしてログインした後、POS画面から直接パスワードを確認・再生成できます。

1. 画面右上の **「☰」（メニューボタン）** をタップ
2. **「🔑 パスワード確認」** をタップ
3. 現在のオーナー用・スタッフ用パスワードが表示されます

### パスワードを変更したい場合

同じ画面の **「再生成」** ボタンをタップすると、新しいパスワードが自動生成されます。

> **注意：** 再生成すると古いパスワードはすぐに使えなくなります。
> 新しいパスワードをスタッフに伝え直してください。

---

## iPad（スタッフ端末）の設定

スタッフ用のiPadには **スタッフ用パスワード** でログインしてください。
スタッフ用でログインすると、以下の制限がかかります：

- 売上分析・Zレポートが見られない
- 料金設定・給与設定が変更できない
- パスワードの確認・再生成ができない
- ロール（権限）の切り替えができない

---

## よくあるトラブル

### 「パスワードが正しくありません」と表示される

- 大文字・小文字を間違えていないか確認してください（例：`vA6KN` の `A` は大文字）
- 日本語入力（IME）がオンになっていないか確認してください
- 5回間違えるとロックされます。1分待ってから再度お試しください

### パスワードがわからなくなった

- サーバーのフォルダにある `.pos_credentials` ファイルを確認してください
- このファイルを削除してサーバーを再起動すると、新しいパスワードが自動生成されます

### スタッフにパスワードを教えたい

- **スタッフ用パスワードのみ** を教えてください
- オーナー用パスワードは管理者以外に教えないでください
